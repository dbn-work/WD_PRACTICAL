Introduction to HTML5

HTML5 is the latest version of HyperText Markup Language, the standard markup language for creating web pages. It introduces new semantic elements, improved forms, and better multimedia support.

What is HTML5?

  • Semantic Elements: Better structure with meaningful tags
  • Multimedia Support: Native audio and video elements
  • Form Enhancements: New input types and validation
  • API Integration: Canvas, geolocation, and storage APIs

Basic HTML5 Document

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First HTML5 Page</title> </head> <body> <h1>Welcome to HTML5!</h1> <p>This is a basic HTML5 document.</p> </body> </html>

HTML Document Structure

Every HTML5 document follows a specific structure with essential elements that define the document type, language, and metadata.

<!DOCTYPE html>

Declares the document as HTML5. Must be the first line in your HTML document.

<html>

Root element that contains all other HTML elements. Use lang attribute for language.

<head>

Contains metadata about the document that isn't displayed on the page.

<body>

Contains all visible content that appears on the web page.

Essential Head Elements

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="Page description"> <title>Page Title</title> <link rel="stylesheet" href="styles.css"> </head>

Semantic HTML5 Tags

Semantic elements clearly describe their meaning to both the browser and developer, improving accessibility and SEO.

<header>

Defines header content for a document or section. Usually contains navigation, logos, or introductory content.

<nav>

Defines navigation links. Use for main site navigation, breadcrumbs, or table of contents.

<main>

Defines the main content area. Should be unique and contain the primary content of the page.

<section>

Defines a section in a document. Use for thematically related content with a heading.

<article>

Defines independent, self-contained content like blog posts, news articles, or comments.

<aside>

Defines content aside from main content. Use for sidebars, related links, or advertisements.

<footer>

Defines footer content. Usually contains copyright, contact info, or related links.

<figure>

Groups media content with its caption. Use with <figcaption> for images, diagrams, or code.

Semantic Structure Example

<header> <nav> <ul> <li><a href="#home">Home</a></li> <li><a href="#about">About</a></li> </ul> </nav> </header> <main> <section> <h2>Welcome Section</h2> <article> <h3>Article Title</h3> <p>Article content...</p> </article> </section> <aside> <h3>Related Links</h3> <ul> <li><a href="#">Link 1</a></li> </ul> </aside> </main> <footer> <p>© 2025 My Website</p> </footer>

Text & Formatting Tags

HTML provides various tags for structuring and formatting text content with semantic meaning.

Heading Tags

<h1> to <h6>

Define headings from most important (h1) to least important (h6). Use hierarchically.

<h1>Main Title</h1> <h2>Section Title</h2> <h3>Subsection Title</h3> <h4>Sub-subsection Title</h4> <h5>Minor Heading</h5> <h6>Smallest Heading</h6>

Text Formatting Tags

<p>

Defines a paragraph. Most common text container element.

<strong>

Indicates strong importance. Usually displayed as bold text.

<em>

Indicates emphasized text. Usually displayed as italic text.

<mark>

Highlights text. Usually displayed with yellow background.

<small>

Represents side comments or fine print. Displays smaller text.

<del>

Represents deleted text. Usually displayed with strikethrough.

<ins>

Represents inserted text. Usually displayed with underline.

<code>

Represents computer code. Usually displayed in monospace font.

Text Formatting Example

<p>This is a <strong>strong</strong> paragraph with <em>emphasized</em> text.</p> <p>Here's some <mark>highlighted</mark> text and <code>inline code</code>.</p> <p><small>This is fine print text.</small></p> <p>Price: <del>$100</del> <ins>$80</ins></p>

Lists & Navigation

HTML provides different types of lists for organizing content in a structured way.

<ul>

Unordered list. Creates a bulleted list of items.

<ol>

Ordered list. Creates a numbered list of items.

<li>

List item. Used inside ul or ol elements.

<dl>

Description list. Groups terms with their descriptions.

<dt>

Description term. Defines a term in a description list.

<dd>

Description definition. Provides description for a term.

List Examples

<!-- Unordered List --> <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> <!-- Ordered List --> <ol> <li>Learn HTML</li> <li>Learn CSS</li> <li>Learn JavaScript</li> </ol> <!-- Description List --> <dl> <dt>HTML</dt> <dd>HyperText Markup Language</dd> <dt>CSS</dt> <dd>Cascading Style Sheets</dd> </dl>

Links & Anchors

Links are fundamental to the web, allowing navigation between pages and sections.

<a>

Anchor element. Creates hyperlinks to other pages, files, sections, or external resources.

Link Types & Attributes

Link Type Attribute Example
External Link href="URL" <a href="https://example.com">Visit Site</a>
Internal Link href="page.html" <a href="about.html">About Page</a>
Section Link href="#id" <a href="#section1">Go to Section</a>
Email Link href="mailto:" <a href="mailto:user@example.com">Email</a>
Phone Link href="tel:" <a href="tel:+1234567890">Call Us</a>

Complete Link Examples

<!-- External Link with New Tab --> <a href="https://www.example.com" target="_blank" rel="noopener"> Visit Example.com </a> <!-- Internal Navigation --> <a href="about.html">About Page</a> <!-- Section Jump Link --> <a href="#contact-section">Go to Contact Section</a> <!-- Email Link with Subject --> <a href="mailto:contact@example.com?subject=Hello"> Send Email </a> <!-- Download Link --> <a href="document.pdf" download="my-document.pdf"> Download PDF </a>

Tables & Data

Tables are used to display tabular data in rows and columns with proper structure and semantics.

<table>

Defines a table. Container for all table elements.

<thead>

Groups header content in a table.

<tbody>

Groups body content in a table.

<tfoot>

Groups footer content in a table.

<tr>

Defines a table row.

<th>

Defines a header cell. Usually bold and centered.

<td>

Defines a data cell. Contains table data.

<caption>

Provides a table caption or title.

Complete Table Example

<table> <caption>Student Grades</caption> <thead> <tr> <th>Name</th> <th>Subject</th> <th>Grade</th> </tr> </thead> <tbody> <tr> <td>John Doe</td> <td>Math</td> <td>A</td> </tr> <tr> <td>Jane Smith</td> <td>Science</td> <td>B+</td> </tr> </tbody> <tfoot> <tr> <td colspan="2">Average</td> <td>A-</td> </tr> </tfoot> </table>

Forms & Input

Forms allow users to input data and interact with web applications through various input elements.

<form>

Defines a form. Container for form elements with action and method attributes.

<input>

Creates various input fields based on type attribute.

<textarea>

Creates multi-line text input area.

<select>

Creates dropdown selection list.

<option>

Defines options within select element.

<label>

Provides labels for form elements. Improves accessibility.

<button>

Creates clickable buttons with various types.

<fieldset>

Groups related form elements together.

Complete Form Example

<form action="/submit" method="post">
<fieldset> <legend>Personal Information</legend>
<label for="name">Name:</label>
<input type="text" id="name" name="name" required>

<label for="email">Email:</label>
<input type="email" id="email" name="email" required>

<label for="age">Age:</label>
<input type="number" id="age" name="age" min="18" max="100">

<label for="country">Country:</label>
<select id="country" name="country">
<option value="">Select Country</option>
<option value="us">United States</option>
<option value="uk">United Kingdom</option>
</select>

<label for="message">Message:</label>
<textarea id="message" name="message" rows="4"></textarea>

<button type="submit">Submit</button>
</fieldset><form>

HTML5 Input Types

  • text: Single-line text input
  • email: Email address input with validation
  • password: Password input (hidden text)
  • number: Numeric input with min/max
  • date: Date picker
  • time: Time picker
  • url: URL input with validation
  • tel: Telephone number input
  • search: Search input field
  • checkbox: Checkbox for multiple selections
  • radio: Radio button for single selection
  • file: File upload input

Media Elements

HTML5 provides native support for multimedia content including images, audio, and video.

<img>

Embeds images. Requires src and alt attributes for accessibility.

<audio>

Embeds audio content with native browser controls.

<video>

Embeds video content with native browser controls.

<source>

Specifies multiple media resources for audio/video elements.

<track>

Provides text tracks for video elements (subtitles, captions).

<canvas>

Provides drawing surface for graphics via JavaScript.

<svg>

Embeds scalable vector graphics directly in HTML.

<iframe>

Embeds another HTML document within current page.

Media Examples

<!-- Image -->
<img src="image.jpg" alt="Description" width="300" height="200">

<!-- Audio -->
<audio controls>
<source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> Your browser does not support audio. </audio>

<!-- Video -->
<video controls width="640" height="360"> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English"> Your browser does not support video. </video>

<!-- Canvas -->
<canvas id="myCanvas" width="400" height="200"></canvas>

<!-- SVG -->
<svg width="100" height="100"> <circle cx="50" cy="50" r="40" fill="blue" /> </svg>

Attributes & Properties

HTML attributes provide additional information about elements and modify their behavior or appearance.

Global Attributes

These attributes can be used on any HTML element:

Attribute Purpose Example
id Unique identifier for the element id="main-content"
class CSS class names for styling class="button primary"
style Inline CSS styles style="color: red;"
title Additional information (tooltip) title="Click for more info"
lang Language of the element content lang="en"
data-* Custom data attributes data-user-id="123"

Common Element-Specific Attributes

Link Attributes

href: URL destination
target: Where to open link
download: Force download
rel: Relationship to current page

Image Attributes

src: Image source URL
alt: Alternative text
width/height: Dimensions
loading: Lazy loading behavior

Form Attributes

name: Form field name
value: Default value
placeholder: Hint text
required: Mandatory field

Table Attributes

colspan: Span multiple columns
rowspan: Span multiple rows
scope: Header scope
headers: Associated headers

Attribute Examples

<!-- Global Attributes --> <div id="main-content" class="container highlight" title="Main content area" lang="en"> <p data-user-id="123" data-role="admin"> User information with custom data </p> </div> <!-- Element-specific Attributes --> <img src="profile.jpg" alt="User profile photo" width="150" height="150" loading="lazy"> <input type="email" name="email" placeholder="Enter your email address" required autocomplete="email"> <a href="https://example.com" target="_blank" rel="noopener noreferrer">External Link</a> <!-- Table with Spanning --> <table> <tr> <th colspan="2">Name</th> <th rowspan="2">Age</th> </tr> <tr> <td>First</td> <td>Last</td> </tr> </table>

Boolean Attributes

Some attributes are boolean (true/false). Their presence indicates true:

  • required: Form field is required
  • disabled: Element is disabled
  • readonly: Element is read-only
  • checked: Checkbox/radio is checked
  • selected: Option is selected
  • hidden: Element is hidden
  • multiple: Multiple selections allowed
  • autoplay: Media plays automatically
  • controls: Media controls are shown
