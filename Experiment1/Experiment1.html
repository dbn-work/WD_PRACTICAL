Experiment 1

Personal Profile Page using HTML5 Tags

Theory

  1. What are HTML5 semantic tags and how do they improve web accessibility?
  2. Explain the difference between ordered lists (<ol>) and unordered lists (<ul>) in HTML.
  3. How do HTML tables help in organizing tabular data and what are their key components?
  4. What is the importance of proper heading hierarchy (h1-h6) in web documents?

HTML5 Tags Used in This Experiment

<header>

Defines header content for a document or section

<section>

Defines a section in a document

<h1> - <h6>

Heading tags for different levels

<ul> / <ol>

Unordered and ordered lists

<table>

Creates tables for tabular data

<a>

Creates hyperlinks to other pages or resources
